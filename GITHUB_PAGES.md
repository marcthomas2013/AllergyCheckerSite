# Publishing On GitHub Pages

This folder is ready to publish as a GitHub Pages site.

1. Push the repository to GitHub.
2. Open the repository settings on GitHub.
3. Go to **Pages**.
4. Set **Source** to **Deploy from a branch**.
5. Choose the `main` branch and the `/docs` folder.
6. Save the setting.

GitHub Pages will use `docs/index.md` as the home page and `docs/_config.yml` as the site configuration.

The home page uses the screenshots in `docs/AppStoreScreens/iPhone17ProMax/`. If you regenerate the screenshots, keep the same filenames or update the image paths in `docs/index.md`.
